tags

jxbの引数は特殊な動作をします。引数の型によって判定されるので違う型であれば引数の順番は問いません。例えば以下のMethodは同義です。

jxb.p('paragraph','gold','gray',{id:'paragraph_01'})
jxb.p('paragraph',{id:'paragraph_01'},'gold','gray')
jxb.p('paragraph','gold',{id:'paragraph_01'},'gray')

いずれも以下のhtmlに変換されます。

<p style="color:gold;background-color:gray;" id="paragraph_01">paragraph</p>

options(ez:opts)

String - [color,background-color]

colorオプションはタグのstyle.color属性に変換します。background-colorオプションはstyle.background-color属性に変換します

example
jxb.p('paragraph','gold')
jxb.p('paragraph','gold','gray')
jxb.p('paragraph','','gray')

<p style="color:gold;">paragraph</p>
<p style="color:gold;background-color:gray;">paragraph</p>
<p style="background-color:gray;">paragraph</p>

Object - [attr...]

attrオプションはタグの属性に変換します。複数個ある場合は結合し、後の値で上書きされます

example
jxb.p('paragraph',{class:'class_0'})
jxb.p('paragraph',{class:'class_0'},{style:{'background-color':'#ccc'}})
jxb.p('paragraph',{class:'class_0'},{style:{'background-color':'#ccc'}},{class:'class_1'})

<p class="class_0">paragraph</p>
<p class="class_0" style="background-color:#ccc;">paragraph</p>
<p class="class_1" style="background-color:#ccc;">paragraph</p>

Number - [template]

templateオプションは数値で指定します(一部のタグのみ対応しています)複数のstyleを設定済みですので簡単に指定できます

example
jxb.h4('header',1)
jxb.hr(2)
jxb.blockquote('blockquote',3)

<h4 style="border-top:0px;border-right:0px;border-bottom:dashed 2px silver;border-left:0px;color:currentColor;">header</h4>
<hr style="height:0px;border-top:1px dashed currentColor;">
<blockquote style="border-top:0px;border-right:0px;border-bottom:0px;border-left:0px;color:currentColor;background:#eaeaea;padding:0.5em 0.5em 0.5em 2em;border:dotted 1px #2222228f;">blockquote</blockquote>

boolean - [character-reference]

実体参照への変更設定

example
jxb.p('<br>')
jxb.p('<br>',true)
jxb.p('<br>',false)

<p><br></p>
<p>&#x3C;br&#x3E;</p>
<p><br></p>

p(text[,color, background-color, attributes, encode])

pタグ(段落)を生成します。

Parameters:

Name Type Description
text string タグの子要素です

options:

Name Type Description
color string style.color属性に変換します。red等の色指定や#fff等の色番号指定ができます
background-color string style.background-color属性に変換します。red等の色指定や#fff等の色番号指定ができます
attributes object 属性に変換されます
encode boolean 実体参照へエンコードされます

examples:

jxb.p("色の設定","gray","silver")
jxb.p("属性の設定",{"style":{"border":"solid 1px #0000ff"}})
jxb.p("複数の属性を適用できます",{"id":"p_04"},{"class":"paragraph"})

result:

<p style="color:gray;background-color:silver;">色の設定</p>
<p style="border:solid 1px #0000ff;">属性の設定</p>
<p id="p_04" class="paragraph">複数の属性を適用できます</p>

display:

色の設定

属性の設定

複数の属性を適用できます
