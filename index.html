jxb

Javascript exports Browser

Install

npm install jxb --save

or

yarn add jxb

Usage

var note = new jxb();
note.h1('title').h2('sub title').list(['one','two','three']).toBrowser()

result: open in Browser

Get Started

jxbを使うとjavascriptだけでhtmlを書くことができます。
CSSはheadタグ内に自動で直書きされるので別途用意する必要はありません。

大まかな流れは3stepです。

  1. インスタンスをnewする
  2. タグ名と同じMethodでHTMLドキュメントを追記していく
  3. toBrowser()でBrowserに出力する

例1

var note = new jxb();
note.h1('title').h2('sub title').list(['one','two','three']).toBrowser()

ここで簡単に各機能を解説します。

1. Create an instance

var note = new jxb();

いわゆるコンストラクタです。 例1では引数なしですが、自動でgithubフレーバーCSSが適用されています。

数種類のCSSが使えたり、ドキュメントのtitle設定、サイドバーの設定などがコンストラクタから行えます。

2. Add HTML document with the same method as tag name

note.h1('title').h2('sub title').list(['one','two','three'])

h1-h6, a, p, img, hr, brなどの主要なタグに対応しています。ListやTableは配列で要素を指定しますので、javascriptらしい記法で追加していくことができます。

3. Output to Browser

note.toBrowser()

toBrowser()は実行すると自動でBrowserが開くMethodです。他にもhtmlに変換するだけのtoHtml()やfileに書き込むtoFile()があります。

tips

table

行に色を付ける

data行にstyleオブジェクトを入れると行に適用されます。

sample code

var data, note;
data = [
  ['name', 'age', 'score'],
  ['ann', 16, 98, {style: {color: 'red'}}],
  ['bob', 22, 76]
];
note = new jxb();
note.table(data);
note.toBrowser();

result

name age score
ann 16 98
bob 22 76

行に色を付ける(条件分岐)

sample code

data = [
  ['name','age','score'],
  ['ann',16,98],
  ['bob',22,76]
]
data.forEach (row)->
  if row[1] > 20
    row.push({style:{color: 'red'}})
note = new jxb()
note.table(data)
note.toBrowser()

result

name age score
ann 16 98
bob 22 76

セルに色を付ける

jxb.tableCellを使います

sample code

data = [
  ['name','age','score'],
  ['ann',16,98],
  ['bob',jxb.tableCell(22,'gold','gray'),76]
]
note = new jxb()
note.table(data)
note.toBrowser()

result

name age score
ann 16 98
bob 22 76

セルに色を付ける(条件分岐)

sample code

data = [
  ['name','age','score'],
  ['ann',16,98],
  ['bob',22,76]
]
data.forEach (row)->
  if row[0] is 'ann'
    row[0] = jxb.tableCell(row[0],'gold','gray')
note = new jxb()
note.table(data)
note.toBrowser()

result

name age score
ann 16 98
bob 22 76

css

ここではcssを設定する方法を説明します。

templateを読み込む

コンストラクタにnumber型を渡すとjxbに用意してあるcssを読み込みます。

note = new jxb(1)

cssは22種類あります。

  • sindresorhus/generate-github-markdown-css

  • jonathantneal/sanitize.css (0)

  • jasonm23.github.io/markdown-css-themes/ (1-16)
  • markdowncss/retro (17)
  • markdowncss/modest (18)
  • markdowncss/air (19)
  • simonlc/Markdown-CSS (20)

指定のない場合や存在しないtemplateを指定した場合はGithub Markdown CSSが読み込まれます。

cssを書きこむ

自分で書いたcssを使いたい場合はcss文字列をArrayに格納してコンストラクタに渡します。

コンストラクタはArray型をcssだと解釈します 以下も同様で

css = 'a:hover h4{background-color:red; cursor : pointer;}'
note = new jxb([css])

cssを読み込む

cdnを読み込む

複数のcssを設定する

  arr = [
    99999,
    '<link href="https://fonts.googleapis.com/css?family=Gaegu" rel="stylesheet">',
    "p {font-family: 'Gaegu', cursive;}",
    'a:hover h4{background-color:red; cursor : pointer;}'
  ]
  note = new jxb(arr)

constructor([title,css,option])

title: string

Html documentのtitleを設定します。

example

note = new jxb('document title')

css: array

htmlのcssを設定します。

数値型の場合、jxbに組み込みのcssを読み込みます。

example

note = new jxb([11])

httpで始まるstringの場合、head内にlinkタグを生成します。

example

note = new jxb('https://stackpath.bootstrapcdn.com/bootstrap/4.1.0/css/bootstrap.min.css')
// <link href="https://stackpath.bootstrapcdn.com/bootstrap/4.1.0/css/bootstrap.min.css" rel="stylesheet">

linkタグの場合、そのままhead内に展開します。

example

note = new jxb('<link href="https://fonts.googleapis.com/css?family=Ubuntu" rel="stylesheet">"')
// <link href="https://fonts.googleapis.com/css?family=Ubuntu" rel="stylesheet">"

その他の文字列はcssと見なし、styleタグが生成されます。

example

note = new jxb('a{color:red;}')
// '<style type="text/css">a{color:red;}</style>'
