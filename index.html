overview

インスタンスの作成

new jxb()で新しいインスタンスを作成します

note = new jxb()

タグの挿入

インスタンスにはタグ名のメソッドがあります。例えばPタグを挿入する場合は以下のように行います

note = new jxb()
note.p('pタグが生成されます')

出力

タグ等を追加したインスタンスをhtmlとして出力します。

note = new jxb()
note.p('pタグが生成されます')
.open()

openメソッドはインスタンスの内容をhtmlに変換、fileに保存、ブラウザで開くという一連の動作を行います。他に以下のメソッドがあります。

tags

HTMLのタグに相当するメソッドです。

arguments

jxbのMethodには多彩なoptionがあります。例えばpre関数は第一引数が必須となり、他はoptionです。

この関数に以下の様な引数を渡してみましょう。

jxb.pre(
  'paragraph <hr/> ',
  'gold',
  'gray',
  {id:'testId'},
  {class:'testClass'},
  true
)

出力されるhtmlはブラウザでの表示は以下になります。

<pre style="color:gold;background-color:gray;" id="testId" class="testClass">paragraph &lt;hr/&gt; </pre>
paragraph <hr/>

この場合の引数は以下の６つを与えていることになります。

[
  'children (string)'
  'color (string)'
  'background-color (string)'
  'attributes (object)'
  'attributes (object)'
  'character reference (boolean)'
]

次はid="testId"だけを設定してみましょう。どのような引数を設定すればいいのでしょうか。

jxb.pre(
  'paragraph',
  {id:'testId'},
)

これだけです。gold等と指定してあった場所にnullを置く必要はありません。ポイントは引数は型によって判定されるということです。また引数の型が違うなら順番は問いません

それでは最初の例をもう１度、引数の順番を変えて実行してみます。

jxb.pre(
  'paragraph <hr/> ',
  'gold',
  true,
  {id:'testId'},
  {class:'testClass'},
  'gray'
)

出力されるhtmlは全く同じになります。

<pre style="color:gold;background-color:gray;" id="testId" class="testClass">paragraph &lt;hr/&gt; </pre>
paragraph <hr/>

  • 必須の引数は第一引数のchildrenだけで他はoptional
  • 型によって引数が判定される
  • 異なる型なら引数の順番は問わない

color(string)

colorオプションはタグのstyle.color属性に変換します。

example
"black","gold" // style="color:black;
"#222"  // style="color:#222;

background-color(string)

background-colorオプションはタグのstyle.background-color属性に変換します。colorオプションとセットで使います。background-colorのみを設定したい場合はcolorオプションを空文字列にします。

example
"black" ,"gray" // style="color:black;background-color:gray;
"" ,"gray" // style="background-color:gray;

attributes(object)

attributesオプションは属性に変換されます

example
{id: 'testId'} // id="testId"
{id: 'testId',style:{color:'gold'}} // id="testId" style="color:gold;"

複数設定すると結合します。重複する場合は上書きされます

color_gold = {style:{color:'gold'}}
color_silver = {style:{color:'silver'}}
testId = {id:"testId"}
note.pre('test', color_gold, color_silver, testId)
<pre style="color:silver;" id="testId">test</pre>
