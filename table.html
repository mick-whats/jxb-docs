table(data [,width,caption])

tableタグを生成します。引数として2次元配列が必要です。

Parameters:
text(string)リンクのtext
ruby(string)ルビ
[attributes](Object) タグの属性をObjectで指定します
Returns:
(Object)jxbのinstanceが返ります
example:
var tableData1;
tableData1 = [["one", "two", "three"], [1, 2, 3], [4, 5, 6]];
note.table(tableData1, {
  id: "table_01"
});

セルに色を付けるには, tableCellを使います

sample
var ann_name;
ann_name = jxb.tableCell("ann", "gold", "gray");
tableArray = [["name", "age", "score"], [ann_name, 16, 98], ["bob", 22, 76]];
return jxb.table(tableArray);

動的に、行に色を付けるサンプルです

sample
var newArr;
tableArray = [["name", "age", "score"], ["ann", 16, 98], ["bob", 22, 76]];
newArr = tableArray.map(function(a) {
  if (a[1] > 20) {
    a[1] = jxb.tableCell(`${a[1]}`, "", "gold");
  }
  return a;
});
return jxb.table(newArr);

行に色を付けるには,styleを加えます

sample
var row_cyan, row_pink;
row_pink = {
  style: {
    "background-color": "pink"
  }
};
row_cyan = {
  style: {
    "background-color": "cyan"
  }
};
tableArray = [
  ["name", "age", "score"],
  ["ann", 16, 98, row_pink],
  ["bob", 22, 76, row_cyan]
];
return jxb.table(tableArray);
